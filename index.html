Symox ist eine ähnlich wie BASIC aufgebaute Programmiersprache, mit der man nützliche Sachen machen kann.

Installation

Bedingungen:

  • Python 3.10.4
  • Visual Studio Code (und Python Extension)
  • Symox Source Code

    • Anleitung:

    1. Installiere Python 3.10.4 (Link oben)

    2. Installiere Visual Studio Code (Link oben)

    3. Drücke nun in der Leiste links auf Erweiterungen und suche hier nach Python und lade es herunter (oder lade dir die Python-Extension über den Link oben herunter)

    4. Nun lade dir hier den Source Code von Symox herunter

    5. Entzippe die Datei und verschiebe den normalen Ordner (Symox-main) zu den Downloads

    6. Gehe jetzt auf Visual Studio Code und klicke oben Datei an. Wähle hier "Ordner öffnen..." aus.

    7. Suche dann den Ordner Symox-main und klicke auf "Ordner auswählen"

    8. Jetzt wird dir der Source Code von Symox in Visual Studio Code angezeigt. Mache einen Rechtsklick auf shell.py und wähle ganz unten "Python Datei im Terminal ausführen" aus.

    9. Fertig! Gebe nun im Terminal deinen "Code" ein.

    Mathematische Operationen

    In Symox kannst du mathematische Operationen ausführen mit +, -, *, / und ^.

    Schreibe hierfür einfach deine Rechnung in diesem Syntax:

    symox > 9 + 4
    symox > 13

    Du kannst auch mit Dezimalbrüchen rechnen. Beispiel:

    symox > 0.5 - 0.6
    symox > -0.1

    In Symox kannst du auch den Power Operator nutzen. Hierfür benutze ^. Beispiel:

    symox > 9^2
    symox > 81

    Variablen

    Variablen kannst du auch benutzen. Um eine Variable festzulegen schreibe folgendes in das Terminal:

    symox > VAR {var:name} = {value}*

    *{} mit deinen Werten auswechseln

    Jetzt wurde deine Variable auf den angegebenen Wert gesetzt. Um mit der Variable zu rechnen, schreibe einfach ihren Namen und danach das, was du rechnen willst:.

    symox > {var:name}
    symox > {var:name} + 6

    Vergleiche

    In dieser Programmiersprache kannst du auch zwei Zahlen oder Variablen vergleichen. Hierfür gibt es == (gleich), != (nicht gleich), < (kleiner als), > (größer als), <= (kleiner gleich), >= (größer gleich), AND und OR.

    Beispiel:

    symox > 7 == 7
    symox > 1
    symox > 6 >= 9
    symox > 0

    0 und 1 ersetzen false und true.

    If-Statements

    In dieser Programmiersprache gibt es auch If-Statements. Benutze sie, wie im folgenden Beispiel gezeigt:

    symox > IF 5 == 5 THEN 123
    symox > 123
    symox > IF 5 == 6 THEN 123 ELSE 1234
    symox > 1234

    Das ganze geht auch mit Variablen. Beispiel:

    symox > VAR a = 2
    symox > 2
    symox > IF a == 2 THEN 123
    symox > 123
    symox > IF a >= 3 THEN 123 ELSE 1234
    symox > 1234

    Hier gibt es auch das ELIF Statement, dass wenn etwas eine andere bestimmte value hat, etwas anderes printet. Beispiel:

    symox > IF 2 == 3 THEN 123 ELIF 2 == 2 THEN 1234
    symox > 1234

    Hierbei kannst du auch die OR und AND Statements adden. Beispiel:

    symox > VAR a = 2
    symox > 2
    symox > IF a OR 5 == 5 THEN 123
    symox > 123
    symox > IF a AND 2 == 2 THEN 1234
    symox > 1234

    Funktionen

    Mit Funktionen kannst du z.B. Sachen etwas leichter ausrechnen.

    Beispiel:

    symox > FUN {function:name}* (a, b) -> a + b
    symox > <function {function:name}>
    symox > {function:name}(4, 6)
    symox > 10

    *{function:name} mit dem Namen deiner Funktion ersetzen.

    Strings

    Man kann nun auch Strings in Symox nutzen.

    Beispiel:

    symox > "test"
    symox > "test"

    Man kann diese auch zusammenfügen oder multiplizieren.

    Beispiel:

    symox > "Hello " + "World"
    symox > "Hello World"
    symox > "Hello World "* 3
    symox > "Hello World Hello World Hello World"

    Listen

    Nun kann man auch Listen in Symox erstellen. Hierfür, schreibe die Liste in eckigen Klammern und liste sie mit Kommas auf.

    Beispiel:

    symox > [1, 2, 3]
    symox > [1, 2, 3]

    Hierbei kann man auch die Rechenzeichen benutzen.

    Beispiel:

    symox > [1, 2, 3] + 4
    symox > [1, 2, 3, 4]
    symox > [1, 2, 3] - 0
    symox > [2, 3]
    symox > [1, 2, 3] * [3, 4, 5]
    symox > [1, 2, 3, 3, 4, 5]
    symox > [1, 2, 3] / 1
    symox > 2

    Integrierte Funktionen

    In Symox gibt es auch integrierte Funktionen.

    Beispiele:

    symox > PRINT("text")
    symox > test
    symox > VAR name = INPUT()
    Müller
    Müller
    symox > name
    symox > Müller
    symox > VAR age = INPUT_INT*()
    symox > 2
    symox > 2
    symox > age
    symox > 2
    symox > CLEAR()**
    symox > IS_NUM(123)
    symox > 1
    symox > IS_STR("test")
    symox > 1
    symox > IS_LIST([1, 2, 3])
    symox > 1
    symox > VAR list = [1, 2, 3]
    symox > APPEND(list, 4)
    symox > 0
    symox > list
    symox > [1, 2, 3, 4]
    symox > POP(list, 3)
    symox > 4
    symox > list
    symox > [1, 2, 3]
    symox > EXTEND(list, [4, 5, 6])
    symox > 0
    symox > list
    symox > [1, 2, 3, 4, 5, 6]

    *INT = Integers also nur Zahlen

    **Der Text im Terminal wird gecleart

    Multi-Line-Statements

    In Symox kannst du auch Multi-Line-Statements verwenden. Hierfür, benutze einfach ; wie im folgenden Beispiel:

    Beispiel:

    symox > PRINT("Hello World"); PRINT("Test erfolgreich")
    Hello World
    Test erfolgreich

    RETURN, CONTINUE und BREAK

    RETURN, CONTINUE und BREAK sind auch nutzbar. Benutze die Statements wie im folgenden Beispiel:

    Beispiel:

    symox > FOR i = 0 TO 10 THEN; IF i == 4 THEN CONTINUE ELIF i == 8 THEN BREAK; VAR a = a + i; END
    symox > 0
    symox > FUN test(); VAR foo = 5; RETURN foo; END
    symox > <function test>

    In Datei ausführen

    Du kannst nun mit RUN("file") eine Datei, mit Symox Code ausführen.

    Beispiel:

    symox > RUN("example.sy")
    symox > Hello World

    Kommentare

    Mit Kommentaren (#) kannst du deinen Code kommentieren. WICHTIG: Dies geht nur, wenn du deinen Code in einer Extra-Datei ausführst.

    Beispiel:

    #Symox ist cool
    PRINT("Hello")
